## Cách cài đặt project
- Cài đặt composer
- Tải project
- Tải các gói cho project
```
composer insatll
```
-Cấu hình file env để tạo kết nối với database
-Sau đó chạy lệnh sau để tạo database
```
php artisan migrate
```
-Chạy project
```
php artisan serve
```
